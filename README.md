# 🎓 Smart Attendance App — Face Recognition

An AI-powered attendance system that automatically marks student 
attendance using face recognition technology.

## 🔗 Live Demo
[https://smartattendanceapp-c6173.web.app](https://smartattendanceapp-c6173.web.app)

## 📋 Project Overview
A mobile app where faculty captures a class photo and AI automatically 
marks attendance. Late students go through an approval workflow:
**Faculty → HOD → Admin**

## 🛠️ Technologies Used
- **Python** — DeepFace, OpenCV, TensorFlow
- **Firebase** — Firestore database + Hosting
- **FlutterFlow** — No-code mobile app UI
- **Make.com** — Approval workflow automation
- **Google Sheets** — Analytics dashboard
- **Claude AI** — Code generation
